# Used Volkswagen Golf Price Analysis

## Overview
This project analyzes 100 used Volkswagen Golf listings within 10 miles of Birmingham (B4 7ET) to identify the main factors affecting used car prices.

## Objectives
- Understand which variables influence used car prices
- Compare local prices with the national benchmark
- Test relationships between fuel type and efficiency
- Build a regression model for price prediction
- Provide business recommendations for inventory and pricing strategy

## Tools Used
- Excel
- SPSS
- Power BI

## Analysis Performed
- Descriptive analysis
- One-sample t-test
- Chi-square test
- ANOVA
- Correlation analysis
- Multiple linear regression

## Key Findings
- Local Birmingham prices were not significantly different from the national benchmark
- Age and mileage had strong negative relationships with price
- Automatic transmission and larger engines increased price
- Fuel type significantly influenced efficiency and pricing
- The regression model explained about 88% of the variation in price

## Business Recommendations
- Focus on petrol automatics priced between £10,000 and £20,000
- Use diesel for the budget-efficient segment
- Position hybrids as eco-premium options
- Highlight black automatic higher-spec cars as premium listings

## Conclusion
This project demonstrates practical business analytics skills by combining descriptive statistics, hypothesis testing, and predictive modeling to support pricing and inventory decisions in the used car market.
